Pusoy Private Room

Online SPA: 1 banker vs 3 players
Supabase online mode

Create or join a room

Banker creates a room. Players join with the 4-digit code. Each player only receives their own cards before showdown.

Run the SQL setup first in Supabase. Then upload this HTML to Vercel or open it locally for testing.

How it works

Cards and submitted hands are stored in Supabase. Room updates are synced using Supabase Realtime events.

Flow: 1. Banker creates room 2. Kids join with room code 3. Banker starts round 4. Each browser fetches its own cards 5. Players submit Top / Middle / Bottom 6. Banker reveals showdown 7. Everyone sees final hands and scoring
